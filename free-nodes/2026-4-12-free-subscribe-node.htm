什么是 Shadowrocket？

Shadowrocket 是一款专为 iOS 设备（包括 iPhone 和 iPad）设计的网络代理工具，它通过 VPN 技术帮助用户绕过网络限制，访问被屏蔽的网站和应用。它的核心功能包括：

  1. 科学上网：突破地区或网络限制，轻松访问全球内容。
  2. 隐私保护：通过加密网络请求，防止数据泄露或被监控。
  3. 流量监控：实时查看网络流量使用情况，优化数据管理。

Shadowrocket 支持多种代理协议，包括 HTTP/SOCKS5、Shadowsocks、Vmess/VLess 等，使其适用于不同的网络环境和需求。

Shadowrocket 在 iPad 上的使用指南

1. 如何下载 Shadowrocket？

由于 Shadowrocket 在某些地区的 App Store 可能无法直接搜索到，建议使用国际版 App Store 进行下载。具体步骤如下：

  1. 打开 iPad 上的 App Store。
  2. 在搜索栏输入 “Shadowrocket”。
  3. 找到应用后，点击 “获取” 或 “购买”（Shadowrocket 是付费应用）。
  4. 下载完成后，点击 “打开” 进入应用。

注意：如果无法在本地 App Store 找到 Shadowrocket，可以尝试切换至其他地区的 Apple ID 进行下载。

2. 如何配置 Shadowrocket？

成功下载后，接下来需要进行代理服务器的配置。以下是详细步骤：

第一步：添加代理服务器

  1. 打开 Shadowrocket 应用。
  2. 点击右上角的 “+” 按钮，进入配置界面。
  3. 根据你的代理服务提供商提供的信息，填写以下内容：
    • 服务器地址（IP 或域名）
    • 端口号
    • 用户名和密码（如有）
    • 加密方式（如 AES-256-GCM）

第二步：选择 VPN 协议

Shadowrocket 支持多种协议，选择合适的协议能优化连接速度和稳定性：
- HTTP/SOCKS5：适合普通网页浏览，兼容性强。
- Shadowsocks：轻量级代理，适合大多数网站。
- Vmess/VLess：适用于高性能需求，如流媒体解锁。

第三步：保存并连接

  1. 完成配置后，点击 “保存”。
  2. 返回主界面，选择刚刚添加的配置。
  3. 点击 “连接”，等待 VPN 建立成功。

3. 如何测试 Shadowrocket 是否正常工作？

连接成功后，可以通过以下方式验证：
1. 打开 Safari 或其他浏览器，访问 Google、YouTube 等通常受限的网站。
2. 使用 IP 检测工具（如 whatismyip.com）检查当前 IP 是否已切换至代理服务器所在地区。
3. 测试流媒体平台（如 Netflix、Disney+）是否能正常播放。

FAQ

Sing-box 的域名规则如何实现分流？
Sing-box 支持按域名匹配流量，将不同网站或应用流量分配到指定节点。结合策略组自动切换低延迟节点，可优化访问速度，保证网络稳定性并提高整体体验。
VMess 的 TLS 选项何时使用最合适？
TLS 可提供加密和伪装，适合在高封锁或不稳定网络环境使用。结合域名伪装和证书，可让流量看起来像普通 HTTPS 请求，提升抗封锁能力和隐蔽性。
Hysteria2 多路复用如何优化短连接性能？
多路复用允许多个短连接共享同一通道，减少握手延迟和资源消耗。适用于游戏、即时通讯等高频短连接场景，提高吞吐量和连接稳定性，优化整体网络体验。
VMess 节点无法连接时应如何排查？
首先检查服务器是否在线，其次查看端口、防火墙和 TLS 配置是否正确。还可以通过日志观察 handshake 是否成功，或尝试更换传输协议如 WS、gRPC 来确认问题来源。
Sing-box 的性能比 Clash 好吗？
Sing-box 在性能优化上更为出色，特别是在高并发连接下延迟更低。它采用 Go 语言编写并优化了内存占用，比传统 Clash 内核运行更轻盈，尤其适合服务器端部署或移动端设备使用。
WinXray 的多策略组如何优化流量分配？
多策略组允许将节点按用途、延迟或地理位置分类，并结合规则文件自动分流流量。这样不同应用如游戏、视频和网页流量都能走最优节点，提高整体访问速度和稳定性。
Netch 支持游戏加速吗？
Netch 专为游戏加速设计，支持 UDP 转发和 TAP 驱动，可有效降低延迟。用户可以为特定游戏添加路由规则，实现分流加速，从而提升国际游戏体验。
Nekoray 的节点排序依据是什么？
Nekoray 默认按节点延迟排序。用户可以自定义排序规则，如按订阅来源、名称或连接成功率排列，从而快速找到性能最佳或最常用的节点。
VMess 的 gRPC 传输适合移动网络吗？
gRPC 传输支持长连接和多路复用，减少握手次数，降低延迟。对于移动网络和高延迟环境，它能提供稳定和高速的访问体验，同时降低掉线和延迟波动风险。
Hiddify Next 可以兼容旧版 Hiddify 吗？
Hiddify Next 完全向下兼容旧版配置，用户可直接导入原有 JSON 或 YAML 文件。系统会自动检测格式并进行迁移，非常适合升级旧环境到新版客户端。
